A new logging feature is now in place to help diagnosing problems with modding dave tables.

Can be enabled with changing the following in %AppData%/The Creative Assembly/Warhammer3/scripts/preferences.script.txt :
vfs_log_level 1;

Once enabled when launching the game all database errors will output in:
%AppData%/The Creative Assembly/Warhammer3/logs/db_validation.log.txt
